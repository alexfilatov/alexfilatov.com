## Overview

A collection of open-source Elixir libraries solving real problems in blockchain, real-time communication, authentication, and networking.

## Featured Projects

### livekit-elixir-sdk

Elixir server SDK for LiveKit, enabling real-time audio and video communication in Elixir applications.

- Server-side token generation
- Room management APIs
- Webhook handling
- Used in production at Gymbile

[View on GitHub](https://github.com/alexfilatov/livekit-elixir-sdk)

---

### borsh

Elixir implementation of BORSH (Binary Object Representation Serializer for Hashing), the binary serialization format used by NEAR Protocol.

- Efficient binary encoding/decoding
- Full NEAR compatibility
- Type-safe serialization

[View on GitHub](https://github.com/alexfilatov/borsh)

---

### near-api-ex

Library for building NEAR Protocol DApps in Elixir. Enables smart contract interaction and blockchain operations from Elixir applications.

- Account management
- Transaction signing
- Smart contract calls
- RPC client

[View on GitHub](https://github.com/alexfilatov/near-api-ex)

---

### torex

Elixir library for making requests through the Tor network, providing anonymity and privacy for network operations.

- Tor SOCKS proxy support
- Circuit management
- Privacy-focused networking

[View on GitHub](https://github.com/alexfilatov/torex)

---

### guardian_redis

Redis adapter for Guardian authentication library, enabling scalable token storage and management.

- Token blacklisting
- Distributed session management
- High-performance storage

[View on GitHub](https://github.com/alexfilatov/guardian_redis)
